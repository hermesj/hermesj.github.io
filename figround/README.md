# Figure and Ground / Figur und Hintergrund

A literary project: two parallel prose texts on the myth of Odysseus and the Siren, presented as an interleaved visual poster, a text overlay (focus version) and audio overlay (audio version).

- [Demo](https://hermesj.github.io/figround/)
- [Overview](index.html)
- [Figur und Hintergrund](figur-hintergrund.html) — the original German text, poster version
- [Figure and Ground](figure-ground.html) — an independent English rendering, poster version
- [Focus](figure-ground-focus.html) – Interactive reading view — gyroscope, touch, or mouse control the focus and visibility of both texts (de/en).
- [Audio prototype](audio-prototype.html) — experimental audio version (de/en) – work in progress
- [Audio prototype 2](audio-prototype-2.html) — second experimental audio version (de/en) with more features buzt less chique – work in progress
